Welcome to Trovu

This extension makes it easier to use trovu.net in your browser.

It defines keyboard shortcuts to open a new tab or window with the Trovu search console.

Settings

    1. chrome://extensions/.
    2. On the left sidebar, click Keyboard shortcuts.
    3. Find the settings for Trovu.
    1. Go to about:addons.
    2. Click the gear icon ⚙️.
    3. Click Manage Extension Shortcuts.
